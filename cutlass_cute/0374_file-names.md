#### [File Names](https://docs.nvidia.com/cutlass/latest/media/docs/cpp#file-names)[](https://docs.nvidia.com/cutlass/latest/media/docs/cpp/#file-names "Permalink to this headline")

New files should be named using `snake_case`
with extension `.hpp` for header files,
`.cu` for CUDA sources,
and `.cpp` for C++ host-only source files.

Header files with extension `.h`
are CUTLASS 2.x legacy headers.
