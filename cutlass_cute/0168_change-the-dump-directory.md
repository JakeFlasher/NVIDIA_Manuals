### [Change the dump directory](https://docs.nvidia.com/cutlass/latest/media/docs/pythonDSL/cute_dsl_general#change-the-dump-directory)[](https://docs.nvidia.com/cutlass/latest/media/docs/pythonDSL/cute_dsl_general/#change-the-dump-directory "Permalink to this headline")

By default, all dumped files are saved in the current working directory. To specify a different directory for the dumped files, please set the environment variable CUTE_DSL_DUMP_DIR accordingly.
