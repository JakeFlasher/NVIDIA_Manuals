### [Log Categories and Levels](https://docs.nvidia.com/cutlass/latest/media/docs/pythonDSL/cute_dsl_general#log-categories-and-levels)[](https://docs.nvidia.com/cutlass/latest/media/docs/pythonDSL/cute_dsl_general/#log-categories-and-levels "Permalink to this headline")

Similar to standard Python logging, different log levels provide varying degrees of detail:

| Level | Description |
| --- | --- |
| 0 | Disabled |
| 10 | Debug |
| 20 | Info |
| 30 | Warning |
| 40 | Error |
| 50 | Critical |
