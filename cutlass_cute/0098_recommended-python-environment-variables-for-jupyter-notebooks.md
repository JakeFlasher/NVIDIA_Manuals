## [Recommended Python environment variables for jupyter notebooks](https://docs.nvidia.com/cutlass/latest/media/docs/pythonDSL#recommended-python-environment-variables-for-jupyter-notebooks)[](https://docs.nvidia.com/cutlass/latest/media/docs/pythonDSL/#recommended-python-environment-variables-for-jupyter-notebooks "Permalink to this headline")

We recommend setting the following environment variable when running jupyter notebooks.

```bash
export PYTHONUNBUFFERED=1
```
