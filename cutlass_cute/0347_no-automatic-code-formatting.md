### [No automatic code formatting](https://docs.nvidia.com/cutlass/latest/media/docs/cpp#no-automatic-code-formatting)[](https://docs.nvidia.com/cutlass/latest/media/docs/cpp/#no-automatic-code-formatting "Permalink to this headline")

Do not use any kind of automatic code formatting,
like `clang-format`, on CUTLASS code.
