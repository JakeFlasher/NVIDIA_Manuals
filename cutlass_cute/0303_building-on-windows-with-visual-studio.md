# [Building on Windows with Visual Studio](https://docs.nvidia.com/cutlass/latest/media/docs/cpp/build#building-on-windows-with-visual-studio)[](https://docs.nvidia.com/cutlass/latest/media/docs/cpp/build/#building-on-windows-with-visual-studio "Permalink to this headline")

CUTLASS 3.2 reintroduces support for the Microsoft Visual Studio compiler on Windows.
Users and developers may build either
in Visual Studio’s graphical integrated development environment,
or on the command line with `cmake --build`.
