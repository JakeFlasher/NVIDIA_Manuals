### [3.5.3.1. Extended GPU Memory](https://docs.nvidia.com/cuda/cuda-programming-guide/03-advanced#extended-gpu-memory)[](https://docs.nvidia.com/cuda/cuda-programming-guide/03-advanced/#extended-gpu-memory "Permalink to this headline")

[Extended GPU memory](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/extended-gpu-memory.html#extended-gpu-memory) is a feature available in NVLink-C2C connected systems that enables efficient access to all memory within the system from within a GPU. EGM is covered in detail in [Section 4.17](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/extended-gpu-memory.html#extended-gpu-memory).
