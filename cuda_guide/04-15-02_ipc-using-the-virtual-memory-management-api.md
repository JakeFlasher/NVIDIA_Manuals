## [4.15.2. IPC using the Virtual Memory Management API](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics#ipc-using-the-virtual-memory-management-api)[](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/#ipc-using-the-virtual-memory-management-api "Permalink to this headline")

The CUDA Virtual Memory Management API allows the creation of
IPC-shareable memory allocations, and it supports multiple operating
systems by virtue of operating-system specific IPC handle data structures.
