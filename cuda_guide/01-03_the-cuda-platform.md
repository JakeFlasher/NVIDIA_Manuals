# [1.3. The CUDA platform](https://docs.nvidia.com/cuda/cuda-programming-guide/01-introduction#the-cuda-platform)[](https://docs.nvidia.com/cuda/cuda-programming-guide/01-introduction/#the-cuda-platform "Permalink to this headline")

The NVIDIA CUDA platform consists of many pieces of software and hardware and many important technologies developed to enable computing on heterogeneous systems. This chapter serves to introduce some of the fundamental concepts and components of the CUDA platform that are important for application developers to understand. This chapter, like [Programming Model](https://docs.nvidia.com/cuda/cuda-programming-guide/01-introduction/programming-model.html#programming-model), is not specific to any programming language, but applies to everything that uses the CUDA platform.
