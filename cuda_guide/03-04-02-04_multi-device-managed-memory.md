### [3.4.2.4. Multi-Device Managed Memory](https://docs.nvidia.com/cuda/cuda-programming-guide/03-advanced#multi-device-managed-memory)[](https://docs.nvidia.com/cuda/cuda-programming-guide/03-advanced/#multi-device-managed-memory "Permalink to this headline")

Managed memory can be used on multi-GPU systems with peer-to-peer support.
The detailed requirements for concurrent multi-device managed memory access
and APIs for GPU-exclusive access to managed memory are described in [Multi-GPU](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/unified-memory.html#um-legacy-multi-gpu).
