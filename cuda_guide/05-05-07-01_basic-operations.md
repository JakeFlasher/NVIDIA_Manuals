### [5.5.7.1. Basic Operations](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices#basic-operations)[](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices/#basic-operations "Permalink to this headline")

[CUDA Math API](https://docs.nvidia.com/cuda/cuda-math-api/index.html) for basic operations are available in both host and device code, except for `__float128`.

All the following functions have a maximum ULP error of zero.
