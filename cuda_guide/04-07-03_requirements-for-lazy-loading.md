## [4.7.3. Requirements for Lazy Loading](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics#requirements-for-lazy-loading)[](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/#requirements-for-lazy-loading "Permalink to this headline")

Lazy loading is a joint feature of both the CUDA runtime and driver. Lazy loading is only available when the runtime and driver version requirements are satisfied.
