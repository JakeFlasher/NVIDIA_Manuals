## [2.1.2. Kernels](https://docs.nvidia.com/cuda/cuda-programming-guide/02-basics#kernels)[](https://docs.nvidia.com/cuda/cuda-programming-guide/02-basics/#kernels "Permalink to this headline")

As mentioned in the introduction to the [CUDA Programming Model](https://docs.nvidia.com/cuda/cuda-programming-guide/01-introduction/programming-model.html#programming-model), functions which execute on the GPU which can be invoked from the host are called kernels. Kernels are written to be run by many parallel threads simultaneously.
