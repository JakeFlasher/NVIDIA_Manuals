## [5.4.9. Compiler Optimization Hints](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices#compiler-optimization-hints)[](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices/#compiler-optimization-hints "Permalink to this headline")

Compiler optimization hints decorate code with additional information to help the compiler optimize generated code.

- The built-in functions are always available in the device code.
- Host code support depends on the host compiler.
