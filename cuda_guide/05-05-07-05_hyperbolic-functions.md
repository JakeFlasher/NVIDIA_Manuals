### [5.5.7.5. Hyperbolic Functions](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices#hyperbolic-functions)[](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices/#hyperbolic-functions "Permalink to this headline")

[CUDA Math API](https://docs.nvidia.com/cuda/cuda-math-api/index.html) for hyperbolic functions are available in both host and device code only for `float` and `double` types.

***** <sub>Mathematical functions marked with “N/A” are not natively available for CUDA-extended floating-point types, such as __half and __nv_bfloat16. In these cases, the functions are emulated by converting to a float type and then converting the result back.</sub>
