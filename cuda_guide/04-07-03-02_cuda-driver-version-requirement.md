### [4.7.3.2. CUDA Driver Version Requirement](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics#cuda-driver-version-requirement)[](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/#cuda-driver-version-requirement "Permalink to this headline")

Lazy loading requires driver version 515 or newer. Lazy loading is not available for driver versions older than 515, even when using CUDA toolkit 11.7 or newer.
