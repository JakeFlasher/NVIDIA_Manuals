## [5.6.1. Memory Barrier Primitives Interface](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices#memory-barrier-primitives-interface)[](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices/#memory-barrier-primitives-interface "Permalink to this headline")

The primitives API is a C-like interface to `cuda::barrier` functionality. These primitives are available by including the `<cuda_awbarrier_primitives.h>` header.
