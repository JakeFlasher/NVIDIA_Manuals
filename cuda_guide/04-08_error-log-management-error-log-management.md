# [4.8. Error Log Management](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics#error-log-management)[](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/#error-log-management "Permalink to this headline")

The _Error Log Management_ mechanism allows for CUDA API errors to be reported
to developers in a plain-English format that describes the cause of the issue.
