## [5.6.4. CUDA Device Runtime](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices#cuda-device-runtime)[](https://docs.nvidia.com/cuda/cuda-programming-guide/05-appendices/#cuda-device-runtime "Permalink to this headline")

The CUDA device runtime is an API available within kernel code which provides many of the same capabilities as the CUDA Runtime API on the host. These APIs are used most often in the contexts of [CUDA Dynamic Parallelism](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/dynamic-parallelism.html#cuda-dynamic-parallelism) or [Device Graph Launch](https://docs.nvidia.com/cuda/cuda-programming-guide/04-special-topics/cuda-graphs.html#cuda-graphs-device-graph-launch).
