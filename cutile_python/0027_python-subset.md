## [Python Subset](https://docs.nvidia.com/cuda/cutile-python#python-subset)[](https://docs.nvidia.com/cuda/cutile-python/#python-subset "Permalink to this headline")

[Tile code](https://docs.nvidia.com/cuda/cutile-python/#tile-code) supports a subset of the Python language.
Within [tile code](https://docs.nvidia.com/cuda/cutile-python/#tile-code), there is no Python runtime.

Only Python features explicitly enumerated in this document are supported.
Many features, such as lambdas, exceptions, and coroutines are not supported today.
