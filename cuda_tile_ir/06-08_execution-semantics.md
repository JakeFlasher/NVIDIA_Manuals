## [6.8. Execution Semantics](https://docs.nvidia.com/cuda/tile-ir/latest/sections#execution-semantics)[](https://docs.nvidia.com/cuda/tile-ir/latest/sections/#execution-semantics "Permalink to this headline")

**Tile IR** program execution starts with a kernel launch. The kernel launch API is uniform for all CUDA kernels
and is specified in the [CUDA Runtime API](https://docs.nvidia.com/cuda/cuda-runtime-api/index.html).
