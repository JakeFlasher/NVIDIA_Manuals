## [4.1. Primitives](https://docs.nvidia.com/cuda/tile-ir/latest/sections#primitives)[](https://docs.nvidia.com/cuda/tile-ir/latest/sections/#primitives "Permalink to this headline")

The bytecode is composed of a handful of primitive types, each with their own encoding
described below. The format uses these primitives to encode more complex data structures
for representing the full set of **Tile IR** items.
